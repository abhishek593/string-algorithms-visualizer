Assumptions:

  1. Odd Manacher finds the odd length palindromes centered at character.
  2. Even Manacher finds the even length palindromes centered at gap between two characters.
  3. Current rightmost palindrome is shown by sky blue color.
  4. When the current index is withing this rightmost palindrome, we find the mirror point of this index and take this as starting point for current index. This mirror position is shown by Brown color.
